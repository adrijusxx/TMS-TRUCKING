# 📄 PDF Export Guide for Alogix Pitch Deck

This guide shows you how to convert your pitch materials to PDF format for distribution.

---

## 🎯 WHY PDF?

- **Universal compatibility**: Opens on any device
- **Professional**: Can't be easily edited
- **Trackable**: Use DocSend to see who views it
- **Small file size**: Easy to email (<10 MB)
- **Print-ready**: Looks great on paper

---

## METHOD 1: EXPORT FROM HTML (QUICKEST)

### Using Chrome (Recommended):

```bash
1. Open alogix-pitch-deck.html in Chrome
2. Press Ctrl+P (Windows) or Cmd+P (Mac)
3. Configure print settings:
   ✓ Destination: Save as PDF
   ✓ Layout: Landscape
   ✓ Margins: None
   ✓ Scale: 100%
   ✓ Options: ✓ Background graphics
4. Click "Save"
5. Name it: "Alogix-Pitch-Deck.pdf"
```

### Result:
- 15 slides exported as PDF
- Each slide is one page
- Interactive elements become static

### Pros:
✅ Fast (2 minutes)  
✅ Preserves exact styling  
✅ No additional software needed  

### Cons:
❌ Navigation buttons appear in PDF  
❌ Can't edit individual slides  
❌ File size may be larger (~5-10 MB)  

---

## METHOD 2: EXPORT FROM POWERPOINT (BEST QUALITY)

### Using Microsoft PowerPoint:

```bash
1. Open your PowerPoint file
2. File → Export → Create PDF/XPS Document
3. Options:
   ✓ Optimize for: Standard (publishing online and printing)
   ✓ Include: All slides
   ✓ Frame slides: Unchecked
4. Click "Publish"
```

### Using Google Slides:

```bash
1. Open your Google Slides presentation
2. File → Download → PDF Document (.pdf)
3. PDF automatically downloads
```

### Result:
- Professional-quality PDF
- Exact layout from PowerPoint
- Optimized file size (2-5 MB)

### Pros:
✅ Best quality  
✅ Smallest file size  
✅ Easy to edit source before exporting  
✅ Can add notes/speaker notes  

### Cons:
❌ Requires creating PowerPoint first  
❌ Takes longer (3-4 hours total)  

---

## METHOD 3: PRINT TO PDF (MANUAL)

If you want to capture specific pages or sections:

### Windows:

```bash
1. Open the HTML or PowerPoint in fullscreen
2. Press Print Screen for each slide
3. Paste into Microsoft Word
4. File → Save As → PDF
```

### Mac:

```bash
1. Open the file
2. Cmd+P (Print)
3. PDF dropdown → Save as PDF
4. Choose filename and location
```

## 📐 PDF QUALITY SETTINGS

### Resolution:
- **Standard**: 150 DPI (good for email)
- **High**: 300 DPI (good for printing)
- **Web**: 72 DPI (smallest file size)

### Compression:
- **None**: Largest file, best quality
- **JPEG**: Medium file, good quality
- **JPEG 2000**: Small file, acceptable quality

### Recommended Settings:
```
Resolution: 150 DPI
Compression: JPEG (Medium)
Color: RGB
Target file size: 2-5 MB
```

## 🎨 POST-EXPORT OPTIMIZATION

### Compress the PDF:

**Online (Free)**:
1. Go to ilovepdf.com/compress_pdf
2. Upload your PDF
3. Choose "Recommended compression"
4. Download compressed version

**Adobe Acrobat (Paid)**:
1. File → Save As Other → Reduced Size PDF
2. Choose compatibility: Acrobat 10.0 and later
3. Click OK

### Target File Sizes:
- **Email attachment**: < 10 MB (< 5 MB ideal)
- **DocSend**: Any size (they compress automatically)
- **Investor portal**: < 20 MB
- **Print**: Uncompressed (highest quality)

## 📊 USING DOCSEND FOR TRACKING

### Why DocSend:
- Track who views your deck
- See which slides they spend time on
- Get notified when deck is opened
- Expire links after a certain date
- Password-protect decks

### How to Use:

```bash
1. Go to docsend.com (free trial available)
2. Upload your PDF
3. Share → Copy link
4. Customize:
   ✓ Require email to view
   ✓ Set expiration date
   ✓ Add password
   ✓ Disable download
5. Send link to investors
```

### What You'll Learn:
- Who opened the deck
- How long they spent on each slide
- Which slides they revisited
- If they shared with others

### Best Practices:
- Create unique link per investor
- Follow up if they stop at Slide 5 (problem slide)
- If they view multiple times → Very interested!
- If they skip to financials → They're serious

## 🖨️ PRINTING YOUR DECK

### For In-Person Meetings:

**Print Settings**:
- **Paper**: 8.5" x 11" or 11" x 17" (landscape)
- **Color**: Full color (not grayscale)
- **Paper quality**: 32 lb premium (not standard 20 lb)
- **Finish**: Matte (not glossy—easier to read)
- **Binding**: Saddle stitch or comb binding

**Where to Print**:
- **FedEx Office**: Same-day, high quality
- **Staples Print**: Good quality, affordable
- **Local print shop**: Best quality, custom options
- **Online (Vistaprint)**: Cheap, but 3-5 day shipping

**What to Print**:
- 1 full deck for you (speaker notes on back)
- 5-10 copies for investors (no notes)
- 20 one-pagers for distribution

## 🔧 TROUBLESHOOTING

### Problem: PDF file size is 50 MB+
**Solution**: 
- Use ilovepdf.com to compress
- Re-export from PowerPoint with "Standard" quality
- Reduce image resolution to 150 DPI

### Problem: Slides look blurry in PDF
**Solution**:
- Export at higher resolution (300 DPI)
- Use PNG images instead of JPG
- Ensure source images are high-res

### Problem: Can't print to PDF
**Solution**:
- Install Microsoft Print to PDF driver
- Use online converter: smallpdf.com
- Export from Google Slides instead

### Problem: PDF won't open on investor's device
**Solution**:
- Ensure it's a standard PDF (not PDF/A)
- Don't password-protect (causes issues)
- Send via DocSend link instead
